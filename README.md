## Setup
To run this project, do this:

1) Clone project in your PC
```
$ git clone https://github.com/Grayder0152/CourseCatalog.git
```
2) Build docker image:
```
$ docker-compose build --no-cache
```
3) Up docker container:
```
$ docker-compose up
```
